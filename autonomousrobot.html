Mechatronics Autonomous Robot

UCSC - Mechatroncis

Project Description

this is h5

This robot was the result of a 5 week long final project for the class ECE 118 - Intro to Mechatroncis. The project merges software, electrical, mechanical, power, and systems engineering as well as project managment in order to produce a fully functioning autonomous robot capable of traversing a randomly generated field and solving difficult tasks for each field.
